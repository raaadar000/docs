# FramerForge — публичный гайд

FramerForge — внутренний инструмент TOIS для деплоя сайтов из Framer в GitHub в один клик.

## Для кого этот гайд

- Дизайнеры: подготовка проекта во Framer и проверка готовности к деплою.
- Продакт-менеджеры: планирование релизов, контроль статусов, взаимодействие с админами.
- Разработчики: техническое сопровождение, окружение, миграции, диагностика.
- Администраторы: управление проектами, доступами и безопасностью.

## Ограничение по админке

Доступ к `/admin` есть только у ограниченного списка лиц. Остальные пользователи могут работать только через интерфейс деплоя в рамках выданных им проектов.
